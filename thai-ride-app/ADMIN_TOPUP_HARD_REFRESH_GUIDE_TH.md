#### Chrome / Edge

```
Windows: Ctrl + Shift + R
Mac: Cmd + Shift + R
```

#### Firefox

```
Windows: Ctrl + F5
Mac: Cmd + Shift + R
```

#### Safari

```
Mac: Cmd + Option + R
```

## 🎓 ทำไมถึงเกิดปัญหานี้?

### Cache ของเบราว์เซอร์

เบราว์เซอร์เก็บข้อมูลเก่าไว้เพื่อให้โหลดเร็ว แต่บางครั้งทำให้:

- ❌ ไม่เห็นข้อมูลใหม่
- ❌ เห็นข้อมูลเก่า
- ❌ ต้อง refresh ถึงจะเห็นข้อมูลล่าสุด

### เมื่อไหร่ที่ต้อง Hard Refresh?

- ✅ หลังจากลูกค้าเติมเงิน
- ✅ หลังจากแก้ไขข้อมูลในฐานข้อมูล
- ✅ เมื่อข้อมูลดูเก่า/ไม่ตรง
- ✅ เมื่อมีการอัพเดทระบบ

## 💡 เคล็ดลับ

### สำหรับ Admin

1. **เปิด Auto-Refresh** เพื่อดูข้อมูลแบบ Real-time
2. **Hard Refresh** ทุกครั้งที่เข้าหน้าใหม่
3. **ใช้ Incognito Mode** เมื่อต้องการทดสอบ
4. **ดูเวลา** ที่รีเฟรชล่าสุด

### ปุ่มที่มีในหน้า Admin

| ปุ่ม            | ความหมาย                       | เมื่อไหร่ใช้                |
| --------------- | ------------------------------ | --------------------------- |
| 🔄              | รีเฟรชด้วยตนเอง                | เมื่อต้องการดูข้อมูลล่าสุด  |
| รีเฟรชอัตโนมัติ | เปิด/ปิด Auto-refresh          | เมื่อต้องการดูแบบ Real-time |
| ค้นหา           | ค้นหาด้วยชื่อ/เบอร์/เลขอ้างอิง | เมื่อต้องการหาคำขอเฉพาะ     |
