## 🎯 สิ่งที่เปลี่ยนแปลง

เพิ่ม **เลขคำสั่งซื้อ (Tracking ID)** ให้กับการเติมเงินทั้งหมด เพื่อให้ลูกค้าสามารถอ้างอิงและติดตามได้ง่ายขึ้น

---

## 📱 วิธีทดสอบ

### ขั้นตอนที่ 1: เปิดหน้ากระเป๋าเงิน

1. เปิดแอป Thai Ride
2. ไปที่เมนู **"กระเป๋าเงิน"**
3. คลิกที่แท็บ **"เติมเงิน"**

### ขั้นตอนที่ 2: ตรวจสอบ Tracking ID

**สิ่งที่ควรเห็น:**

```
┌─────────────────────────────────────┐
│ ฿500.00              [รอดำเนินการ] │
│ 📋 TOP-20260128-123456              │  ← เลขคำสั่งซื้อ (สีเขียว)
│ 28 ม.ค. 14:30                      │
└─────────────────────────────────────┘
```

**ตรวจสอบ:**

- ✅ เลขคำสั่งซื้อแสดงเป็นสีเขียว
- ✅ มีไอคอน 📋 ด้านหน้า
- ✅ รูปแบบ: `TOP-YYYYMMDD-XXXXXX`
- ✅ แสดงทุกรายการเติมเงิน

### ขั้นตอนที่ 3: ทดสอบคัดลอก

1. **คลิกที่เลขคำสั่งซื้อ** (สีเขียว)
2. **ตรวจสอบ:**
   - ✅ แจ้งเตือน "คัดลอกเลขคำสั่งซื้อแล้ว" ปรากฏ
   - ✅ เลขถูกคัดลอกไปยัง clipboard
3. **วาง (Paste)** ในแอปอื่น เช่น Line, Notes
4. **ตรวจสอบ:** เลขที่วางตรงกับที่แสดงในแอป

### ขั้นตอนที่ 4: ทดสอบ Hover Effect (Desktop)

**ถ้าใช้คอมพิวเตอร์:**

1. เลื่อนเมาส์ไปที่เลขคำสั่งซื้อ
2. **ตรวจสอบ:**
   - ✅ พื้นหลังเปลี่ยนเป็นสีเขียวอ่อนขึ้น
   - ✅ เลขเคลื่อนขึ้นเล็กน้อย
   - ✅ Tooltip แสดง "คลิกเพื่อคัดลอก: TOP-..."

### ขั้นตอนที่ 5: ทดสอบบนมือถือ

**ถ้าใช้มือถือ:**

1. แตะที่เลขคำสั่งซื้อ
2. **ตรวจสอบ:**
   - ✅ แตะได้ง่าย (ไม่ต้องแตะแม่นมาก)
   - ✅ มี feedback เมื่อแตะ
   - ✅ คัดลอกสำเร็จ

## 🐛 ปัญหาที่อาจพบ

### ปัญหา 1: ไม่เห็น Tracking ID

**สาเหตุ:**

- รายการเก่าที่สร้างก่อนอัปเดต
- Browser cache

**วิธีแก้:**

1. Hard refresh: `Ctrl+Shift+R` (Windows) หรือ `Cmd+Shift+R` (Mac)
2. Clear cache
3. สร้างรายการเติมเงินใหม่

### ปัญหา 2: คัดลอกไม่ได้

**สาเหตุ:**

- Browser ไม่รองรับ Clipboard API
- ไม่ได้อนุญาต clipboard permission

**วิธีแก้:**

1. ตรวจสอบ browser version (ควรเป็นเวอร์ชันล่าสุด)
2. อนุญาต clipboard permission
3. ลองใช้ browser อื่น

### ปัญหา 3: Toast ไม่แสดง

**สาเหตุ:**

- CSS conflict
- Z-index issue

**วิธีแก้:**

1. Hard refresh
2. ตรวจสอบ console errors
3. แจ้งทีมพัฒนา

## 📊 ตัวอย่าง Tracking ID

```
TOP-20260128-123456  ← สร้างวันที่ 28 ม.ค. 2026
TOP-20260128-789012  ← วันเดียวกัน เลขต่างกัน
TOP-20260129-456789  ← วันถัดไป
```

**รูปแบบ:**

- `TOP` = Top-up (เติมเงิน)
- `20260128` = วันที่ (YYYYMMDD)
- `123456` = เลขสุ่ม 6 หลัก
